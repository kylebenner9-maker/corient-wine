One-time Setup

Judgment of Paris V

To enable multiplayer (everyone on their own phone), you need a free Firebase database. This takes about 5 minutes. Follow the steps below, then paste your config into the HTML file.

Step-by-Step Firebase Setup

1Go to console.firebase.google.com and sign in with a Google account.
2Click "Create a project". Give it any name (e.g. judgment-of-paris). Disable Google Analytics if prompted.
3In the left sidebar, click Build → Realtime Database. Click "Create database". Choose "Start in test mode". Click Enable.
4In the left sidebar, click the ⚙️ gear → Project settings. Scroll to "Your apps" and click the </> Web icon. Register the app (any nickname). Copy the firebaseConfig object.
5Open judgment-of-paris-v2.html in a text editor. Find the section that says PASTE YOUR FIREBASE CONFIG BELOW near the top and replace the placeholder values with your config values.
6Save the file and re-upload it to Netlify (drag-and-drop the file onto your existing Netlify site dashboard).

Security Note

The database starts in "test mode" which allows open read/write — fine for a one-evening event. After the event, go to Firebase → Realtime Database → Rules and set both .read and .write to false to lock it down.

Where to find your config

Your config will look like this — replace all the placeholder values in the HTML:

const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "myproject.firebaseapp.com", databaseURL: "https://myproject-default-rtdb.firebaseio.com", projectId: "myproject", storageBucket: "myproject.appspot.com", messagingSenderId: "123456789", appId: "1:123:web:abc123" };
